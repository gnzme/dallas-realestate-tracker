# Dallas Real Estate Market Tracker 🏠

An interactive real estate dashboard built with Python and Streamlit, using live property data from the Zillow API via RapidAPI. Tracks for-sale and for-rent listings across Dallas, TX neighborhoods.

🔗 **Live app:** [dallas-realestate-tracker.streamlit.app](https://dallas-realestate-tracker.streamlit.app)

---

## Features

- Live data from Zillow API — 1,000+ real Dallas property listings
- Interactive filters: listing type, property type, price range, days on market, zip code
- Property map with color-coded pricing by location
- 5 analytical charts: price distribution, average by zip, price vs sqft, days on market, price cuts
- Listings table with direct links to Zillow property pages
- Photo gallery with property details and one-click Zillow access
- For Sale and For Rent support

## Tech Stack

- Python 3.x
- Streamlit — interactive web dashboard
- Plotly Express — interactive charts and map
- Pandas — data manipulation and analysis
- Requests — Zillow API via RapidAPI
- Hosted on Streamlit Cloud

## Project Structure

```
dallas-realestate-tracker/
├── app.py            # Streamlit dashboard (main app)
├── scraper.py        # Zillow API scraper — fetches real listings
├── analysis.py       # Static chart generation (PNG outputs)
├── quick_test.py     # API connection test
├── requirements.txt  # Python dependencies
├── data/             # CSV datasets (auto-generated by scraper)
│   ├── dallas_listings_YYYYMMDD.csv   # For Sale
│   └── dallas_rentals_YYYYMMDD.csv    # For Rent
└── outputs/          # Static chart PNGs (generated by analysis.py)
```

## Setup & Local Development

**1. Clone the repo**
```bash
git clone https://github.com/gnzme/dallas-realestate-tracker.git
cd dallas-realestate-tracker
```

**2. Install dependencies**
```bash
pip install -r requirements.txt
```

**3. Configure API key**

Create a `.env` file in the root directory:
```
RAPIDAPI_KEY=your_rapidapi_key_here
```

Get a free key at [rapidapi.com](https://rapidapi.com) — subscribe to the **private-zillow** API (free tier: 250 requests/month).

**4. Fetch data**
```bash
python scraper.py
```

**5. Run the dashboard**
```bash
streamlit run app.py
```

## Data Pipeline

```
Zillow API (RapidAPI)
        ↓
   scraper.py          # fetches up to 1,000 listings per run
        ↓
data/dallas_listings_YYYYMMDD.csv
        ↓
    app.py             # loads CSV, cleans data, renders dashboard
```

## Key Insights from the Data

- Dallas median home price: ~$647k (For Sale)
- 38%+ of listings have had at least one price reduction
- Condos and Townhomes sell significantly faster than Single Family homes
- Zip codes 75209 and 75225 consistently rank as the most expensive areas
- Zip code 75223 (East Dallas) shows the highest rate of price cuts
